Hidemaru mail growl
===================

ライセンス
----------

確認中

秀丸メール用マクロ
------------------

メール受信時に Growl でタイトルを通知する。

使用方法
--------

マクロを登録する

    マクロ > マクロ登録 > 自動起動 > 受信が一段落した時

不具合
------

  * 新規メールが既読になってしまうことがある
  * 受信済みメール判定をする際に該当のメールがあるフォルダへ移動するので、受信前後で選択していたメールが変わることがある
